# 🎓 Tổng kết năm nhất – Trần Gia Bảo

Đây là **website cá nhân tổng kết năm nhất đại học** của mình – **Trần Gia Bảo**, sinh viên ngành **Công Nghệ Thông Tin**.

Trang web được xây dựng bằng **HTML, CSS và JavaScript** nhằm giới thiệu:

* Thông tin cá nhân
* Hành trình học tập năm nhất
* Thành tích học tập và nghiên cứu khoa học
* Thông tin liên hệ

Project này cũng là một **bài thực hành web cá nhân** và một **cột mốc ghi lại hành trình năm nhất đại học**.

# ▶️ Cách chạy project

Đây là **website tĩnh**, không cần server.

### Cách 1 (đơn giản)

Mở file:

```
index.html
```

bằng trình duyệt:

* Google Chrome
* Microsoft Edge
* Firefox

---

### Cách 2 (khuyến khích – VS Code)

1. Cài extension **Live Server**
2. Chuột phải vào `index.html`
3. Chọn

```
Open with Live Server
```
